Project Overview

Challenges & Solutions

Challenge: Designing a weapon system that was both modular and easy for team members to extend.

Solution: Created a Blueprint-based parent weapon class with configurable parameters (fire rate, damage, ammo capacity) that allowed designers to create weapon variants without programming knowledge. This accelerated content creation and empowered the entire team.

Challenge: Coordinating development across multiple team members working on interconnected systems.

Solution: Established clear communication protocols through daily standups, used Perforce for version control, and created comprehensive documentation for all shared systems. This minimized merge conflicts and kept everyone aligned.

What I Learned

This project reinforced the importance of clear communication and documentation in team environments. Working with programmers and artists taught me how to design systems that are both technically sound and accessible to non-technical team members.

I also gained valuable experience in agile methodologies, learning how to break large features into manageable tasks, prioritize work effectively, and adapt to changing requirements. The collaborative nature of this project strengthened my ability to give and receive constructive feedback.
