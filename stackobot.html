Project Overview

Stack O Bot is a mechanic-focused design project that explores physics-based interactions and refined control schemes through rapid prototyping and iterative playtesting. The project emphasized creating intuitive, responsive gameplay mechanics that feel satisfying to players.

As a solo developer, I was responsible for all aspects of design and implementation, from initial concept through final polish. This project allowed me to deeply explore Blueprint visual scripting and physics systems in Unreal Engine while maintaining a tight development cycle.

My Contributions

  • Designed core gameplay mechanics and implemented them using Blueprint visual scripting
  • Developed physics-based interactions that respond naturally to player input
  • Refined control schemes through rapid prototyping and iterative testing
  • Balanced difficulty progression through data-driven playtesting and analysis
  • Created responsive feedback systems to enhance player understanding of mechanics
  • Implemented collision detection and physics constraints for stable object stacking
  • Optimized Blueprint performance to maintain smooth frame rates

Challenges & Solutions

Challenge: Creating physics-based stacking that felt responsive yet stable.

Solution: Implemented custom physics constraints and damping values to balance realistic physics with playable game feel. Added subtle visual and audio feedback to help players understand when objects were properly aligned and stable.

Challenge: Designing controls that were intuitive for new players but allowed mastery.

Solution: Created a layered control scheme with basic inputs for beginners and advanced techniques for experienced players. Playtesting with diverse skill levels helped refine the learning curve to be accessible yet deep.

What I Learned

This project deepened my understanding of game feel and the importance of responsive, tactile mechanics. I learned how subtle details like timing windows, input buffering, and visual feedback can dramatically impact player satisfaction and perceived responsiveness.

Working with Unreal Engine's physics system taught me valuable lessons about balancing realism with playability. I discovered that "realistic" physics often need to be tuned and exaggerated to create the most engaging gameplay experience.
